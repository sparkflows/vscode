# Data Profiling Nodes

## General
- **Graph Week Day Distribution** `02-Data-Profiling/GraphWeekDayDistribution.json` — This node Finds the distribution of Week Days from Date values.
- **Graph Year Distribution** `02-Data-Profiling/GraphYearDistribution.json` — This node Finds the distribution of Years from Date values.
- **Histogram** `02-Data-Profiling/Histogram.json` — Computes a histogram of the data using number of bins evenly spaced between the minimum and maximum of the specific columns.
- **Columns Cardinality** `02-Data-Profiling/columnsCardinality.json` — Distribution of categorical data.
- **Correlation** `02-Data-Profiling/correlation.json` — calculates the correlation between two series of data.
- **Correlation** `02-Data-Profiling/correlation_python.json` — calculates the correlation between two series of data.
- **Cross Tab** `02-Data-Profiling/crosstab.json` — Categorical V.
- **Distinct Values In Column** `02-Data-Profiling/distinctvalues.json` — Executes the Distinct Values In Column operation.
- **Flag Outlier** `02-Data-Profiling/flagoutlier.json` — Flag the outlier based on the selected column using Box-and-Whisker technique.
- **Graph Month Distribution** `02-Data-Profiling/monthDistributionGraph.json` — This node Finds the distribution of months from Date values.
- **MultiFlagOutliers** `02-Data-Profiling/multiFlagOutlier.json` — This checks if values fall between Inter Quartile Range.
- **MultiFlagOutliers** `02-Data-Profiling/multiFlagOutlier_python.json` — This checks if values fall between Inter Quartile Range.
- **Null Values In Column** `02-Data-Profiling/nullValuesInColumn.json` — Number of Null Values in Selected Columns.
- **Skewness And Kurtosis** `02-Data-Profiling/skewnessandkurtosis.json` — Executes the Skewness And Kurtosis operation.
- **Summary Statistics** `02-Data-Profiling/summary.json` — Summary statistics provide useful information about sample data.
- **WhatIf Summary Statistics** `02-Data-Profiling/whatIfSummary.json` — WhatIf Summary statistics provide useful information about sample data.
