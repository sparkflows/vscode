# Data Visualization Nodes

## General
- **Bubble Chart** `07-Data-Visualization/BubbleChart.json` — A bubble chart is used to visualize a data set with two to four dimensions.
- **Geo Chart** `07-Data-Visualization/GraphRegionGeo.json` — This node maps the values on a Map.
- **Graph Values** `07-Data-Visualization/GraphValues.json` — Executes the Graph Values operation.
- **Hierarchy Chart** `07-Data-Visualization/HierarchyChart.json` — Executes the Hierarchy Chart operation.
- **Pie Chart** `07-Data-Visualization/PieChart.json` — Executes the Pie Chart operation.
- **Box Plot** `07-Data-Visualization/boxplot.json` — Executes the Box Plot operation.
- **Gauge Chart** `07-Data-Visualization/gauge.json` — Executes the Gauge Chart operation.
- **Graph Group By Column** `07-Data-Visualization/graphgroupbycolumn.json` — Groups the data by the given column and plots the number of records in each group.
- **Bar Chart** `07-Data-Visualization/nodeBarChart.json` — Executes the Bar Chart operation.
- **Bar Group Chart** `07-Data-Visualization/nodeBarGroupChart.json` — Executes the Bar Group Chart operation.
- **Print Rich Text** `07-Data-Visualization/print-rich-text.json` — Displaying rich text.
- **Print N Rows** `07-Data-Visualization/printnrows.json` — Prints the specified number of records in the DataFrame.
- **Scatter Chart** `07-Data-Visualization/scatterChart.json` — Executes the Scatter Chart operation.
- **Scatter Cluster Chart** `07-Data-Visualization/scatterClusterChart.json` — Executes the Scatter Cluster Chart operation.
- **Graph Subplots** `07-Data-Visualization/subplots.json` — Executes the Graph Subplots operation.
- **Word Cloud Chart** `07-Data-Visualization/wordCloudChart.json` — Executes the Word Cloud Chart operation.
